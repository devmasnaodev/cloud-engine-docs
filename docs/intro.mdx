# Cloud Engine

:::warning
**Projeto em estágio inicial:** o Cloud Engine ainda está em desenvolvimento ativo e **não está pronto para uso em produção**. Utilize apenas para testes, avaliação e ambientes controlados.
:::

## O que é o Cloud Engine?

O Cloud Engine é uma plataforma voltada para desenvolvedores, agências e autônomos que precisam gerenciar engines WordPress em VPS — como EasyEngine, WordOps ou similares. O objetivo é centralizar o provisionamento de servidores, as operações das engines e o ciclo de vida dos sites em uma interface unificada.

A plataforma pode ser utilizada em ambiente local ou em nuvem e foi pensada para eliminar a necessidade de acesso SSH constante para tarefas rotineiras de administração.

## Como funciona?

O Cloud Engine **não é instalado diretamente no servidor**. Ele atua como uma camada de gerenciamento remoto que se comunica com o servidor onde sua engine está rodando — funcionando como um painel de controle externo.

Isso significa que, a qualquer momento, você ainda pode acessar o servidor diretamente via SSH normalmente. O Cloud Engine complementa o seu fluxo de trabalho, sem substituir o acesso direto.
